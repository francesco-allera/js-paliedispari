# EXERCISE 24 - BOOLEAN CAREERS TRAINEE:


### Languages used:

- HTML5
- CSS3
- JAVASCRIPT ES5


### Purpose of the exercise:

Create a function to tell if the word is palindrome.

The user chooses odd or even and a number from 1 to 5.

We generate a random number (always from 1 to 5) for the computer.

We add the two numbers together and declare who won.
